# 🏏 AI Sports Analytics Assistant (Ollama + Python)

An AI-powered sports analytics system that answers natural language questions about IPL match data using **Ollama LLM + Pandas**.

---

## 🚀 Features

- Ask questions in natural language
- AI converts questions into structured JSON
- Pandas executes real data analysis
- Works fully offline using Ollama
- Handles IPL match dataset (CSV)

---

## 🧠 Example Questions

- Which team has most wins?
- How many matches were played?
- Which team won most matches?
- Count total matches
